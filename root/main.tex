**Theorem.** The set of all natural numbers $n$ that equal the sum of the factorials of their own decimal digits is exactly $\{1, 2, 145, 40585\}$.

More precisely, define $f(n) = \sum_{d \,\in\, \text{digits}(n)} d!$, where $\text{digits}(n)$ denotes the list of base-10 digits of $n$. Then
$$\{ n \in \mathbb{N} \mid n = f(n) \} = \{1, 2, 145, 40585\}.$$
